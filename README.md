# KoHack26-
# Kehilati: An app focusing on providing information about physical accessibility in different synagogues for users who have physical special needs. Includes ratings on reviews for specific features including entrances and exits, Shabbat elevators, ramps, wheelchair seating, etc. Additionally includes a Google Maps component that guides the user to the accessible entrances when not made obvious.

#Main: Backend js code which feed to a front end HTML code. This part is where you search for shuls, see reviews and gain information. The rest in ancillary to this.

#Login: This part is as described, where you create accounts and log in. It is a simple username and password system. 

#Kehilati Logo: just our symbol

#Accounts: Simply storage of all accounts for login purposes. The other forms of data associated with the accounts, like the comments, are not tied up here

#Reviews: This is JSON storage where we show the reviews, who they are tied to, and all information regarding the reviews and ratings

#Server: This is the technical bit where we use servers to connect what is done accross accounts for a unified page.
